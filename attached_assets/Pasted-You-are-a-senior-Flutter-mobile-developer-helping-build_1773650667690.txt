Core Features to Implement

1. Camera Dashcam Recording

* Use the phone camera to record continuous driving video
* Video should record in loop mode (like dashcams)
* Example: save 5-minute clips continuously

2. GPS Location Tracking

* Collect GPS coordinates periodically
* Send location updates to backend API
* Used for fleet tracking on dashboard

3. Sensor Data Collection
   Collect data from phone sensors such as:

* Accelerometer
* Gyroscope

These will later be used by the AI system to detect:

* Harsh braking
* Sudden acceleration
* Crash events

4. Event Video Upload
   When a risky event is detected (or manually triggered):

* Save the video clip
* Upload it to Firebase Storage
* Send metadata to backend API

Metadata example:

* timestamp
* driver_id
* vehicle_id
* GPS location
* event_type
* video_url

5. Emergency Alert Button
   Add a large **SOS button** inside the app.

When pressed:

* Save last 30 seconds of video
* Upload the clip
* Send emergency alert to backend
